1 Introducción a la programación y Python

¿Qué es la programación?

La programación es la actividad de escribir instrucciones precisas que una computadora puede ejecutar para resolver un problema. Esas instrucciones se escriben en un lenguaje de programación. En esta materia vas a usar Python.

¿Qué es un algoritmo?

Un algoritmo es una secuencia finita y ordenada de pasos para resolver un problema. Un algoritmo puede existir sin computadoras: una receta de cocina es un algoritmo.

Ejemplo cotidiano
Algoritmo "hacer un café":
1. Poner agua en la pava.
2. Encender el fuego.
3. Esperar a que hierva.
4. Poner café en la taza.
5. Echar el agua caliente.

Si seguís los pasos en orden, obtenés café. Eso es un algoritmo.

¿Qué es un programa?

Un programa es un algoritmo escrito en un lenguaje que la computadora entiende. La programación consiste en traducir la idea (el algoritmo) al código del lenguaje.

Python: un primer vistazo

Python es un lenguaje de programación diseñado para ser legible y claro. Se usa para análisis de datos, inteligencia artificial, desarrollo web, ingeniería y mucho más.

print("Hola, mundo")

Este es el programa más chico: le dice a la computadora que muestre en pantalla el texto Hola, mundo. La función print() muestra cosas.

Google Colab

Para esta materia vas a usar Google Colab: un entorno web donde escribís y ejecutás Python sin instalar nada. Lo abrís desde el navegador con tu cuenta de Google.

Variables

Una variable es un nombre que guarda un valor. Pensá en una caja etiquetada: la etiqueta es el nombre, el contenido es el valor.

edad = 25
nombre = "Ariel"
pi = 3.14159

print(edad)
print(nombre)
print(pi)

Se usa el signo = (llamado asignación) para guardar un valor en una variable. Después podés usar el nombre donde antes iba el valor.

Reglas para nombres de variables
• Pueden tener letras, números y _.
• NO pueden empezar con número: 2edad ❌
• NO pueden tener espacios ni acentos: edad_actual ✓, edad actual ❌
• Distinguen mayúsculas: Edad y edad son distintas.
• Elegí nombres descriptivos: total_pagado es mucho mejor que x.

El intérprete de Python

Python es un lenguaje interpretado: la computadora va leyendo el código línea por línea y ejecutándolo al momento. En Colab, cada celda es un bloque de código que podés ejecutar con Shift+Enter.

Errores típicos al empezar
• Olvidar las comillas en los textos: nombre = Ariel ❌ (sin comillas Python busca una variable llamada Ariel), nombre = "Ariel" ✓
• Confundir = con ==: el primero asigna, el segundo compara.
• Olvidar los paréntesis de print().

Funciones

Una función es un bloque de código con nombre que hace una tarea específica. Podés reutilizarlo las veces que quieras.

La estructura básica:

def nombre_funcion(parametros):
    # código de la función
    return resultado
Ejemplo
def doble(x):
    return x * 2

print(doble(5))    # 10
print(doble(7))    # 14
print(doble(-3))   # -6

Partes de una función

  • def: palabra clave que indica que vas a definir una función.
  • Nombre: cómo llamás a tu función.
  • Parámetros: los datos que recibe (entre paréntesis). Pueden ser varios separados por comas o ninguno.
  • Cuerpo: las líneas indentadas debajo (con 4 espacios o un tab). La indentación es obligatoria en Python.
  • return: devuelve el resultado. Una función puede no tener return (devuelve None).
Indentación: lo más importante de Python
A diferencia de otros lenguajes, Python usa indentación (espacios al principio de la línea) para marcar qué código está "dentro" de una función, un if, un ciclo, etc. Siempre usá la misma cantidad de espacios (4 es estándar). Mezclar tabs y espacios causa errores.

Comentarios

Texto que Python ignora. Sirve para explicar qué hace el código.

3 Estructuras de control

¿Qué son las estructuras de control?

Por defecto, Python ejecuta el código de arriba hacia abajo, línea por línea. Las estructuras de control cambian ese flujo: permiten tomar decisiones o repetir acciones.

Condicional: if · muy frecuente en parciales

Ejecuta un bloque de código solo si se cumple una condición.

if edad >= 18:
    print("Sos mayor de edad")

Si edad >= 18 es verdadero, ejecuta el print. Si no, no hace nada y sigue.

if / else

"Si se cumple X, hacé una cosa; si no, hacé otra".

if edad >= 18:
    print("Mayor de edad")
else:
    print("Menor de edad")

if / elif / else

Para varias opciones encadenadas. elif significa "else if":

nota = 7

if nota >= 8:
    print("Muy bien")
elif nota >= 6:
    print("Bien")
elif nota >= 4:
    print("Aprobado")
else:
    print("Desaprobado")

Python evalúa las condiciones en orden. Cuando encuentra una verdadera, ejecuta solo ese bloque y salta todo el resto.

Error clásico
Olvidar los : al final de la condición, o la indentación del bloque interno. Ambos dan errores de sintaxis.

Condicionales anidados

Un if adentro de otro if:

if tiene_dni:
    if edad >= 16:
        print("Puede votar")
    else:
        print("Muy joven para votar")
else:
    print("Necesita DNI")

Ciclo while · muy frecuente

Repite un bloque de código mientras se cumpla una condición.

i = 0
while i < 5:
    print(i)
    i = i + 1

Imprime 0, 1, 2, 3, 4. El ciclo sigue mientras i < 5. En cada vuelta se suma 1, hasta que i llega a 5 y la condición se vuelve falsa.

Casos de uso típicos de while
• Validar datos: "pedir un número hasta que sea positivo".
• Repetir hasta una señal: "jugar hasta que el usuario diga que basta".
• Cuando no sabés de antemano cuántas veces vas a iterar.
¡Cuidado con los ciclos infinitos!
Si la condición nunca se hace falsa, el ciclo nunca termina. Asegurate de que algo dentro del ciclo modifica la variable de la condición.

Validar entrada con while

n = int(input("Ingresá un positivo: "))
while n <= 0:
    print("Tiene que ser positivo")
    n = int(input("Ingresá un positivo: "))

Ciclo for · muy frecuente

Recorre los elementos de una secuencia (lista, string, range...) uno por uno.

for letra in "Python":
    print(letra)

Imprime P, y, t, h, o, n.

nombres = ["Ana", "Bruno", "Carla"]
for nombre in nombres:
    print("Hola", nombre)

La función range()

Genera una secuencia de enteros. Muy usada con for.

FormaGenera
range(5)0, 1, 2, 3, 4
range(2, 7)2, 3, 4, 5, 6
range(0, 10, 2)0, 2, 4, 6, 8 (paso 2)
range(10, 0, -1)10, 9, 8, ..., 1 (cuenta regresiva)
Ojo: range NO incluye el extremo final
range(5) va de 0 a 4 (no incluye el 5). range(2, 7) va de 2 a 6 (no incluye el 7).
Ejemplo: sumar los primeros 10 números
suma = 0
for i in range(1, 11):
    suma = suma + i
print(suma)   # 55

break y continue

  • break: corta el ciclo inmediatamente.
  • continue: salta a la próxima iteración sin ejecutar el resto del cuerpo.
for i in range(10):
    if i == 5:
        break       # corta al llegar a 5
    print(i)

4 Listas y tuplas

¿Qué es una lista?

Una lista es una colección ordenada y mutable de elementos. Se escribe entre corchetes y los elementos se separan con comas.

numeros = [10, 20, 30, 40, 50]
palabras = ["casa", "árbol", "auto"]
mezcla = [1, "hola", 3.14, True]   # puede tener distintos tipos

Acceso por índice

Se accede a los elementos con índice, que empieza desde 0.

numeros = [10, 20, 30, 40, 50]
print(numeros[0])   # 10 (primero)
print(numeros[2])   # 30 (tercero)
print(numeros[-1])  # 50 (último, índice negativo)
print(numeros[-2])  # 40 (anteúltimo)
Índices negativos
En Python, los índices negativos cuentan desde el final: -1 es el último, -2 el anteúltimo, etc. Súper útil y elegante.

Longitud: len()

numeros = [10, 20, 30, 40, 50]
print(len(numeros))   # 5

Slicing (rebanadas)

Para obtener una sublista, usás lista[inicio:fin]. El fin no se incluye.

numeros = [10, 20, 30, 40, 50]
print(numeros[1:4])    # [20, 30, 40]
print(numeros[:3])     # [10, 20, 30] (desde el principio)
print(numeros[2:])     # [30, 40, 50] (hasta el final)
print(numeros[:])      # [10, 20, 30, 40, 50] (copia completa)

Mutabilidad: podés modificar una lista

nums = [10, 20, 30]
nums[1] = 99
print(nums)   # [10, 99, 30]

Recorrer una lista

Dos formas típicas:

# Por elemento
for n in numeros:
    print(n)

# Por índice
for i in range(len(numeros)):
    print(i, numeros[i])

Operadores útiles: in, +, *

5 in [1, 2, 3, 5]       # True (¿está 5?)
"casa" in ["auto","casa"]  # True

[1, 2] + [3, 4]     # [1, 2, 3, 4] (concatena)
[0] * 5             # [0, 0, 0, 0, 0] (repite)

Tuplas: como listas pero inmutables

Una tupla es como una lista, pero no se puede modificar después de crearla. Se escriben con paréntesis:

coord = (3, 4)
colores = ("rojo", "verde", "azul")

print(coord[0])    # 3
# coord[0] = 99    # ERROR: las tuplas son inmutables
¿Cuándo usar tupla en vez de lista?
Cuando los datos no tienen que cambiar: coordenadas, fechas, datos constantes. Es más seguro y más rápido.

Desempaquetado

Podés asignar varios valores a la vez:

x, y = (3, 4)
print(x)   # 3
print(y)   # 4

# También funciona con listas
nombre, edad = ["Ariel", 30]

5 Diccionarios

¿Qué es un diccionario?

Un diccionario es una colección de pares clave:valor. A diferencia de una lista (donde accedés por posición), en un diccionario accedés por una clave que vos elegís.

persona = {"nombre": "Ariel", "edad": 30, "ciudad": "Buenos Aires"}

print(persona["nombre"])   # "Ariel"
print(persona["edad"])     # 30

Sintaxis

  • Llaves { } (no corchetes).
  • Cada par tiene la forma clave: valor.
  • Los pares se separan con comas.
  • Las claves suelen ser strings o números. Los valores pueden ser cualquier cosa.

Crear, agregar, modificar

# Vacío
persona = {}

# Agregar o modificar
persona["nombre"] = "Ariel"
persona["edad"] = 30
persona["edad"] = 31    # modifica (misma clave)

print(persona)   # {'nombre': 'Ariel', 'edad': 31}

Borrar entradas

del persona["edad"]
# o
persona.pop("edad")

¿Existe una clave?

if "nombre" in persona:
    print("Tiene nombre:", persona["nombre"])
Ojo
Si accedés con persona["clave_que_no_existe"], Python tira error. Para evitarlo:
• Usá in antes de acceder.
• O usá persona.get("clave"): devuelve None si no existe en lugar de tirar error.

Métodos útiles

MétodoDevuelve
d.keys()Todas las claves
d.values()Todos los valores
d.items()Pares (clave, valor)
d.get(k, default)Valor o default si no existe

Recorrer un diccionario

persona = {"nombre": "Ariel", "edad": 30}

# Por clave
for clave in persona:
    print(clave, "→", persona[clave])

# Por pares
for clave, valor in persona.items():
    print(clave, "→", valor)

Listas de diccionarios · muy frecuente

Estructura común para datos estructurados. Cada elemento de la lista es un diccionario.

6 Archivos y manejo de errores

¿Por qué trabajar con archivos?

Cuando un programa termina, se pierden todas las variables. Para persistir datos (guardarlos), hay que escribirlos en un archivo. Después los podés leer en otra ejecución.

Abrir un archivo: open()

archivo = open("datos.txt", "r")

El segundo parámetro es el modo:

ModoQué hace
"r"Lectura (read). Falla si no existe.
"w"Escritura (write). Sobrescribe si existe.
"a"Agregar al final (append).
"r+"Lectura y escritura.
Cuidado con "w"
Abre el archivo y borra todo su contenido. Si querés agregar sin perder lo anterior, usá "a".

Leer un archivo

# Leer todo como un solo string
archivo = open("datos.txt", "r")
contenido = archivo.read()
archivo.close()

# Leer línea por línea
archivo = open("datos.txt", "r")
for linea in archivo:
    print(linea)
archivo.close()

Escribir en un archivo

archivo = open("salida.txt", "w")
archivo.write("Hola\n")
archivo.write("Segunda línea\n")
archivo.close()
Salto de línea: \n
Si querés escribir en varias líneas, tenés que agregar \n al final. write() NO lo hace solo.

La forma recomendada: with

Abre el archivo y lo cierra automáticamente al terminar el bloque.

with open("datos.txt", "r") as archivo:
    for linea in archivo:
        print(linea.strip())

strip() quita espacios y saltos de línea al principio y al final.

Manejo de errores

Un error (o excepción) ocurre cuando algo no sale bien: un archivo no existe, el usuario ingresa texto donde esperabas número, dividís por cero, etc.

Por defecto, cuando ocurre un error, el programa se corta. Para manejarlo:

try / except

try:
    n = int(input("Ingresá un número: "))
    print(n * 2)
except ValueError:
    print("Eso no fue un número")

try: "intentá esto".
except: "si falla, hacé esto otro".

Excepciones más comunes

ExcepciónCuándo ocurre
ValueErrorTipo incorrecto: int("hola")
ZeroDivisionErrorDivisión por cero
FileNotFoundErrorArchivo inexistente al abrir
KeyErrorClave no existe en diccionario
IndexErrorÍndice fuera de rango en lista
TypeErrorTipo incompatible: "a" + 5

Múltiples except

try:
    archivo = open("datos.txt", "r")
    n = int(archivo.readline())
    resultado = 10 / n
except FileNotFoundError:
    print("No existe el archivo")
except ValueError:
    print("El archivo no tiene un número")
except ZeroDivisionError:
    print("No se puede dividir por cero")

Validar entrada de usuario (clásico de parcial)

def entPositivo(cartel):
    pedir = True
    while pedir:
        try:
            n = int(input(cartel))
            if n > 0:
                pedir = False
            else:
                print("Tiene que ser mayor que 0")
        except ValueError:
            print("Tiene que ser entero")
    return n

num = entPositivo("Ingresá un natural: ")

7 Pandas (análisis de datos)

¿Qué es una biblioteca?

Una biblioteca (library) es código que otros escribieron y podés usar en tus programas importándolo. Evita "reinventar la rueda". Python viene con muchas bibliotecas incluidas, y podés instalar miles más.

import pandas as pd
import matplotlib.pyplot as plt

import carga la biblioteca. as le da un nombre más corto (alias) para usarla.

¿Qué es Pandas?

Pandas es la biblioteca estándar para trabajar con datos en forma tabular (como planillas Excel) en Python. Se usa muchísimo en ingeniería, ciencia de datos y análisis.

Estructura principal: DataFrame

Un DataFrame es una tabla con filas y columnas. Cada columna es una Series.

import pandas as pd

# Desde un diccionario
datos = {
    "nombre": ["Ana", "Bruno", "Carla"],
    "edad":   [25, 30, 28],
    "ciudad": ["CABA", "Rosario", "CABA"]
}
df = pd.DataFrame(datos)
print(df)

Genera una tabla con columnas nombre, edad, ciudad.

Cargar datos desde un CSV

df = pd.read_csv("datos.csv")

CSV (Comma-Separated Values) es el formato más común: archivo de texto donde cada fila es una línea y las columnas están separadas por comas.

Explorar un DataFrame

MétodoQué muestra
df.head()Primeras 5 filas
df.tail()Últimas 5 filas
df.shape(filas, columnas)
df.columnsNombres de columnas
df.info()Resumen: tipos, no nulos
df.describe()Estadísticas básicas de columnas numéricas

Acceder a columnas

df["edad"]          # la columna edad como Series
df[["nombre","edad"]]   # varias columnas (doble corchetes)

Filtrar filas

# Personas mayores de 27
df[df["edad"] > 27]

# Personas de CABA
df[df["ciudad"] == "CABA"]

# Combinado (con paréntesis)
df[(df["edad"] > 25) & (df["ciudad"] == "CABA")]
En pandas se usa &, |, ~
Con DataFrames NO usás and, or, not, sino &, |, ~. Y siempre envolvé cada condición con paréntesis.

Operaciones útiles

df["edad"].mean()       # promedio
df["edad"].max()        # máximo
df["edad"].min()        # mínimo
df["edad"].sum()        # suma total
df["ciudad"].value_counts()   # cuántos de cada valor

# Ordenar
df.sort_values("edad")            # ascendente
df.sort_values("edad", ascending=False)  # descendente

# Agrupar
df.groupby("ciudad")["edad"].mean()   # promedio de edad por ciudad

Agregar / modificar columnas

df["mayor_de_edad"] = df["edad"] >= 18
df["edad_doble"] = df["edad"] * 2

8 Matplotlib (gráficos)

¿Qué es Matplotlib?

Matplotlib es la biblioteca estándar para crear gráficos en Python. Permite hacer gráficos de línea, barras, dispersión, histogramas, etc.

import matplotlib.pyplot as plt

plt es el alias convencional.

Gráfico de línea básico

x = [1, 2, 3, 4, 5]
y = [2, 4, 1, 5, 3]

plt.plot(x, y)
plt.show()

plt.plot(x, y) arma el gráfico. plt.show() lo muestra.

Agregar títulos y etiquetas

plt.plot(x, y)
plt.title("Mi primer gráfico")
plt.xlabel("Tiempo (s)")
plt.ylabel("Velocidad (m/s)")
plt.show()

Conectar Pandas con Matplotlib

Podés graficar directamente desde un DataFrame:

import pandas as pd
import matplotlib.pyplot as plt

df = pd.read_csv("ventas.csv")
plt.plot(df["mes"], df["total"])
plt.title("Ventas mensuales")
plt.show()

# O incluso más directo
df.plot(x="mes", y="total", kind="bar")
plt.show()
